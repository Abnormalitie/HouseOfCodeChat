README:

Log in is only working with google login, the autologin does not work.
When logging in for the first time, you will be taken to the Settings activity, input name and status, profile image is optional.

Profile picture cannot be set for some reason so everyone will have the placeholder image.

To create a group and start messaging, click the icon in the top right corner and click "create group", input group name and the group will appear in the list.
click the group you want to chat in and from there you can send text and images from the gallery.
